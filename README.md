# Demir Cashless Reports

Отдельный сервис для сотрудников банка. Загружает из Vendotek/TMS клиентов проекта `bank-demir` и формирует отчеты только по безналичным операциям за выбранный период.

## Запуск

1. При необходимости скопируйте `.env.example` в `.env`.
2. Vendotek-логин и пароль уже заданы в коде как дефолт. `.env` нужен только для переопределения порта, проекта или доступов.
3. Установите зависимости:

```bash
npm install
```

4. Запустите сервис:

```bash
npm start
```

По умолчанию приложение откроется на `http://localhost:3100`.

Сотрудникам банка не нужно вводить Vendotek-доступы: сервис сразу открывает экран отчетности и использует встроенный доступ к `bank-demir`.
